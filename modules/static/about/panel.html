What is a module?
    A module is a unit with a specific function. It may contain one HTML file or a group of files (HTML, JavaScript, and CSS) which can be hosted on any static web server under any individual account.
What is a web application?
    A web application is a layout module with a group of other modules linked together.
What is a layout module?
    A layout module is the first module loaded into a web browser. It contains a header (banner), a contents container and a footer.
What is a contents container?
    A contents container is a html div where other modules can be loaded into it.

What is the advantage of a modular distributed web application?
    Fast and flexible.
    Modules can be hosted anywhere and maintained by different people with different skills.
    Modules can be reused in different applications.
    Building a web application is just creating a layout module and links all other functional modules needed together.
